# yfile
simple file manager written in C that runs in the terminals, no colour support required
## keys
| Key         | Action                |
| ----------- | --------------------- |
| Ctrl + Q    | Quit                  |
| Ctrl + A    | Toggle hidden files   |
## features
- [x] file view
- [x] file size view
- [ ] folder size view
- [ ] file operations
- [ ] folder operations
